# helloworld.go

The hello world app in the Go language 

The file `helloworld.go` contains the following:

```
package main

func main() {
	print("hello, world\n")
}
```

It can be run using:

    go run helloworld.go

Or compiled to a binary:

    go build helloworld.go

The binary can be execute using:

    ./helloworld.go
